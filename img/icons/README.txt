PERSONALIZACIÓN (OPCIONAL):
----------------------------
Si deseas usar íconos personalizados, puedes:

1. Reemplazar los archivos SVG existentes con tus propios diseños
2. O agregar archivos PNG (128x128px o 256x256px) con los mismos nombres:
   - instagram.png
   - tiktok.png
   
   Luego actualiza el HTML para usar .png en lugar de .svg

RECOMENDACIONES:
----------------
- Formato PNG con fondo transparente
- Color blanco o negro (se aplicarán filtros CSS)
- Tamaño mínimo: 128x128 píxeles
- Tamaño máximo: 512x512 píxeles
